# BMI 計算機

##  專案簡介 
一個能掌握健康數據的超強工具！

##  功能特色

* 新增特殊的防呆機制(防止沒有輸入身高或體重的情況下就計算BMI)
* 有令人驚豔的背景視覺效果
* 跑馬燈展示

##  使用方式
輸入身高體重，點擊計算，就可以得知BMI了
